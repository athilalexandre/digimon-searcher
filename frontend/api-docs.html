API – Digimon Searcher

Base URL: http://localhost:3000

Endpoints

  • GET /digimons?page=<num>&limit=<num> – Lista paginada
  • GET /digimons/:nome – Busca por nome (case-insensitive e tolerante a acentos)
  • GET /digimons/nivel/:nivel – Filtra por nível
  • GET /digimons/tipo/:tipo – Filtra por tipo
  • GET /digimons/busca?nome=&nivel=&tipo=&atributo=&campo=&page=&limit= – Busca flexível
  • GET /digimons/:nome/detalhes-wikimon – Enriquecimento sob demanda a partir do Wikimon (descrição, imagem e técnicas)

Exemplos rápidos

GET /digimons?page=1&limit=12
GET /digimons/Agumon
GET /digimons/nivel/Adult
GET /digimons/tipo/Vaccine
GET /digimons/busca?nome=garuru&page=1&limit=8

Observação: esta API é local e usa dados derivados da Digi-API e enriquecidos de forma best-effort pelo Wikimon.
